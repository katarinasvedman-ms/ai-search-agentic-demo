# Private 5G Manufacturing Blueprint

## Solution summary

The manufacturing blueprint combines Aurora RAN 6651 radios, Nimbus Indoor 2400 small cells, WaveSync 8600 transport, and CoreSphere packet core automation into a staged private 5G rollout. The design supports machine vision, guided vehicles, and low-latency industrial control patterns.

## Retrieval-friendly comparison notes

Aurora provides outdoor and yard coverage, sector capacity, and wide-area mobility. Nimbus provides hall, corridor, and office coverage with precise indoor zoning. WaveSync handles timing and fronthaul aggregation, while CoreSphere provides lifecycle automation and upgrade control.

## Protected deployment notes

The protected blueprint includes spectrum assumptions, resilient backhaul zoning, timing alarm thresholds, upgrade coordination, and cutover sequencing for AGV and machine-vision workloads. These details are intended for authenticated customers and internal delivery teams.
